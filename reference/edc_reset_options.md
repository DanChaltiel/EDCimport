# Reset all `EDCimport` options.

Reset all `EDCimport` options.

## Usage

``` r
edc_reset_options(
  except = c("edc_lookup", "trialmaster_pw", "path_7zip"),
  quiet = FALSE
)
```

## Arguments

- except:

  options that are not reset by default

- quiet:

  set to `TRUE` to remove the message.

## Value

Nothing, called for its side effects
